Polysun Plugin Interfaces
=========================

This project contains the interfaces required for Polysun plugin controller development.

Contained are:

- IPluginController interface for plugin controllers
- ControllerPlugin interface for plugin detection

The PolysunDemoPlugin and the PolysunPublicPlugin are examples.

This code is Open Source, see LICENSE.txt. It can be used to create new plugin controllers for use in Polysun.

Requirements for controller plugin development
----------------------------------------------

- JDK 7 (http://www.oracle.com/technetwork/java/javase/downloads/java-archive-downloads-javase7-521261.html)
- ant (http://ant.apache.org/bindownload.cgi)

Recommended:

- Eclipse (http://www.eclipse.org/downloads/)

Build
-----

ant


Development
-----------

The Eclipse project can be imported with File > Import... > Existing projects into Workspace.
